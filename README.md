# 🚀 Employee Management System

A modern and responsive **Employee Management System** built using **React.js** and **Tailwind CSS**.  
This project helps admins manage employees, assign tasks, and track progress through a clean and user-friendly dashboard.

---

## 📌 Features

✅ Admin Login Authentication  
✅ Employee Task Management  
✅ Create & Assign Tasks  
✅ Track Task Status (Completed / Pending / Active)  
✅ Responsive Dashboard UI  
✅ Modern and Clean Design  
✅ Fast and Smooth User Experience

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **State Management:** React Hooks
- **Build Tool:** Vite / CRA _(based on your setup)_
